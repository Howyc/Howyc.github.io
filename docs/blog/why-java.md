# 从前端到全栈：我为什么开始学 Java

作为一个 React / TypeScript 开发者，学 Java 的契机很简单——想真正理解后端在做什么。

前端调 API 的时候，总觉得后端是个黑盒。`POST /api/users` 发出去，数据就存进去了，但中间发生了什么？Spring 是怎么把 JSON 变成 Java 对象的？JPA 是怎么自动生成 SQL 的？

带着这些问题，我开始了这个项目。

## 收获

- 理解了三层架构（Controller / Service / Repository）和前端的对应关系
- 搞懂了 `@Autowired` 依赖注入背后的逻辑
- 发现 Java 和 TypeScript 其实非常像，学习曲线没想象中那么陡
