# 项目

## fetch-mcp-demo

一个展示前后端完整数据流的 Demo 项目。

- 前端：React 19 + TypeScript + Vite
- 后端：Spring Boot 3 + JPA + SQLite
- 功能：用户/帖子的 CRUD，JSONPlaceholder 数据同步

**技术亮点**
- 统一 `ApiResponse<T>` 响应格式
- `Optional` 链式处理 404 场景
- `@Transactional` 批量操作原子性保证
